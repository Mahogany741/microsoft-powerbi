Big Data Analytics x Machine Learning 

    Big Data Analytics: (análise Descritiva)

        Extrair conhecimento a partir dos dados (Big Data), aplicando técnicas de análise (Data Science).

    Machine Learning (análise Preditiva)

        Treinar algoritmos para encontrar os padrões nos dados de forma automática e então fazer previsões.

O que é o Power BI 

    Data view (Ferramenta de visualização de dados)
    Preencher a lacuna entre os dados e a tomada de decisão

Power BI Desktop x Power BI Pro x Power BI Premium

Drill-Down (ou granularidade)

    Exemplo: Uma base dados, que você você gera uma coluna baseada na data, ela provavelmente trará o ano na coluna. Isso ocorre pq o PowerBI agrupou pelo ano. Ao clicar nas 'duas setinhas' viradas para baixo, você pode alterar a hierarquia, o que mudará a forma que o PowerBI agrupou... (ano para trimestre por exemplo)

Transformações de dados

    No exemplo utilizado no curso, a coluna preco, estava no formato americano, separando com '.' ao inves de ','. 
    A solução recomendada, é fazer esse tratamento, antes de carregar o arquivo.

    arquivo > opcoes > opcoes > arquivo atual > configuracoes regionais > localidade > ingles (estados unidos)
